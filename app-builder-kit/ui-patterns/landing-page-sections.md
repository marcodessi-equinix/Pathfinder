# UI Pattern Landing Page Sections

## Typische Reihenfolge

1. Hero
2. Kernnutzen
3. Features oder Leistungen
4. Social Proof oder Trust
5. CTA
6. Footer

## Gute Hero-Sektion

- klares Wertversprechen
- kurze Unterzeile
- sichtbarer Haupt-CTA
- optional ein sekundarer CTA

## Gute Trust-Signale

- Logos
- Kennzahlen
- Testimonials
- Garantien oder Sicherheitsmerkmale

## Vermeiden

- zu viele gleichwertige Botschaften im ersten Screen
- CTA erst weit unten zeigen
- mobile Reihenfolge ignorieren
