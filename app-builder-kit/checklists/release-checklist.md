# Release Checklist

- Tut die Aenderung wirklich das Gewuenschte?
- Gibt es offensichtliche Regressionen?
- Sind neue Fehlerfaelle bedacht?
- Ist die UI auf Desktop und mobil brauchbar?
- Sind wichtige Entscheidungen dokumentiert?
- Sind bekannte Grenzen offen benannt?
- Wurden relevante Tests oder manuelle Checks gemacht?
