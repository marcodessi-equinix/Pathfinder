# App Type Landing Page

## Typische Ziele

- klares Wertversprechen
- starke erste Sektion
- wenige, klare Handlungsziele

## Empfohlene UI-Richtung

- markante Hero-Sektion
- starke Hierarchie
- klarer Call to Action
- begrenzte Anzahl von Sections

## Typische Sections

- Hero
- Problem oder Nutzen
- Features oder Leistungen
- Social Proof oder Trust
- CTA
- Footer

## Gute Defaults

- Above-the-fold muss schon die Kernbotschaft erklaeren
- CTA mehrfach, aber nicht chaotisch wiederholen
- mobile Reihenfolge frueh mitdenken
