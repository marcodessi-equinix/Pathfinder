# UI Pattern Dashboard Patterns

## Gute Dashboards

- wichtigste KPI zuerst
- klare Gruppierung nach Themen
- Diagramme nur mit echtem Informationswert
- Tabellen und Filter logisch zusammensetzen

## Gute Hierarchie

- Ebene 1: wichtigste Kennzahlen
- Ebene 2: Trends oder Vergleiche
- Ebene 3: Tabellen, Details, Drill-down

## Vermeiden

- jede Karte gleich stark gestalten
- zu viele Farben fuer neutrale Daten
- wichtige Warnungen optisch untergehen lassen
