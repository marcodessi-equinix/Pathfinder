# Pre Build Checklist

- Ist klar, ob es eine neue App oder eine Anpassung ist?
- Ist der Nutzer oder die Zielgruppe benannt?
- Sind Must-have und Nice-to-have getrennt?
- Ist Version 1 sinnvoll klein geschnitten?
- Gibt es eine erkennbare UI-Richtung?
- Sind technische Grenzen bekannt?
- Ist der erste Umsetzungsabschnitt klein genug?
