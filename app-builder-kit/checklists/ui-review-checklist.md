# UI Review Checklist

- Ist die visuelle Hierarchie sofort lesbar?
- Gibt es genau erkennbare Hauptaktionen?
- Sind Farben klar verteilt und nicht zufaellig?
- Ist der Kontrast fuer Text ausreichend?
- Sind Focus-States sichtbar?
- Funktioniert die UI auf mobilen Breiten sinnvoll?
- Sind Formulare klar beschriftet?
- Haben interaktive Elemente genug Groesse?
- Ist die Oberflaeche produktpassend statt generisch?
